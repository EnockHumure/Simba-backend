## Product Fields Explained

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | Auto | UUID generated by database |
| `name` | string | Yes | Product name (displayed to users) |
| `slug` | string | Yes | URL-friendly identifier (unique) |
| `description` | string | Yes | Full product description |
| `shortDescription` | string | No | Brief description for listings |
| `price` | number | Yes | Current selling price (in Rwf) |
| `comparePrice` | number | No | Original price (for showing discount) |
| `sku` | string | No | Stock keeping unit (unique) |
| `stock` | number | Yes | Available quantity |
| `lowStockAlert` | number | Yes | Threshold for low stock warning |
| `images` | string[] | Yes | Array of image URLs |
| `categoryId` | string | Yes | Reference to category |
| `tags` | string[] | Yes | Search/filter tags |
| `isFeatured` | boolean | Yes | Show in featured section |
| `isActive` | boolean | Yes | Visible to customers |
| `isAlcohol` | boolean | Yes | Age restriction flag |
| `weight` | number | No | Product weight |
| `unit` | string | No | Weight/volume unit (kg, ml, g) |
| `viewCount` | number | Auto | Number of views |
| `salesCount` | number | Auto | Number of sales |
| `rating` | number | Auto | Average rating (0-5) |
| `reviewCount` | number | Auto | Total reviews |
| `createdAt` | datetime | Auto | Creation timestamp |
| `updatedAt` | datetime | Auto | Last update timestamp |

## Example API Calls

### Get All Products
```bash
GET /api/products?page=1&limit=20&categoryId=xxx&search=cola
```

### Get Featured Products
```bash
GET /api/products/featured
```

### Get Product by ID
```bash
GET /api/products/clx1a2b3c4d5e6f7g8h9i0j1
```

### Create Product
```bash
POST /api/products
Content-Type: application/json

{
  "name": "New Product",
  "slug": "new-product",
  "description": "Description",
  "price": 1500,
  "stock": 100,
  "categoryId": "xxx",
  "images": ["/path/to/image.jpg"],
  "tags": ["tag1"],
  "isFeatured": false,
  "isActive": true,
  "isAlcohol": false,
  "lowStockAlert": 10
}
```

### Update Product
```bash
PUT /api/products/clx1a2b3c4d5e6f7g8h9i0j1
Content-Type: application/json

{
  "price": 1600,
  "stock": 150
}
```

### Delete Product
```bash
DELETE /api/products/clx1a2b3c4d5e6f7g8h9i0j1
```
